# LiveAssist Landing Page - Design Notes

## Design Philosophy: Modern, Accessible, Action-Oriented

**Core Aesthetic:**
- Clean, professional interface with emphasis on clarity and usability
- Asymmetric layout with strategic visual hierarchy
- Warm, inviting color palette with strong accent colors for CTAs
- Generous whitespace to reduce cognitive load
- Smooth micro-interactions and transitions

**Color Palette:**
- Primary: Vibrant Blue (#0066FF) - trust, technology, communication
- Secondary: Warm Orange (#FF6B35) - energy, action, fitness
- Accent (Cooking): Warm Green (#4CAF50) - culinary, natural
- Accent (Fitness): Energetic Red (#E74C3C) - intensity, movement
- Accent (Accessibility): Purple (#9B59B6) - inclusivity, care
- Neutral: Slate grays for text and backgrounds
- Light backgrounds with dark text for accessibility

**Typography:**
- Display: Bold, modern sans-serif (Poppins or similar) for headings
- Body: Clean, readable sans-serif (Inter or similar) for content
- Hierarchy: Clear size and weight differentiation

**Layout Approach:**
- Hero section with asymmetric design (image on right, content on left)
- Feature cards with hover effects and icons
- Mode showcase with interactive elements
- Clear CTA buttons throughout
- Responsive grid for mobile-first design

**Key Sections:**
1. Navigation bar (sticky, minimal)
2. Hero section with app preview mockup
3. Problem/Solution section
4. Three mode cards (Cooking, Fitness, Accessibility)
5. Features grid
6. Technology stack showcase
7. CTA section for download
8. Footer with links

**Accessibility First:**
- High contrast ratios for all text
- Large, clickable buttons
- Keyboard navigation support
- Clear focus states
- Semantic HTML structure
